## Tech Stack
- Next.js (Pages Router)
- TypeScript
- Wagmi
- Viem
- Hardhat
- Fhenix Devnet (FHEVM)

## Features
- Wallet connection using injected provider (MetaMask)
- Frontend connected to Fhenix Devnet
- Foundation for private voting logic using FHEVM

## Getting Started

### Install dependencies
npm install

### Run frontend
cd frontend
npm run dev

Open http://localhost:3000
